# Готовые рекламные тексты

Перед публикацией вставь свои ссылки из команды `/promo`.

```

Запустили бот, где можно быстро найти собеседника из твоего города.

- анкета 1 минуту
- лайки и мэтчи
- сразу ссылка в личку при взаимной симпатии

18+.
Попробовать: <ССЫЛКА_ИЗ_/promo>
```

```
Надоело сидеть одному? Есть быстрый бот для знакомств и общения в твоем городе.

Нажал старт -> анкета -> подбор -> мэтч.

18+.
```

## 2) Текст для TikTok/Reels/Shorts (описание)

## 3) Сообщение админу канала (для размещения)
```
Привет! Есть рабочий бот для знакомств/поиска компании (18+), локальная тематика.
Могу дать готовый короткий пост + уникальную ссылку для вашего канала.

Готов обсудить взаимопиар или фикс за размещение.
```

## UTM-логика внутри бота
- Telegram каналы: `src_tg_channels`
- TikTok: `src_tiktok`
- Instagram: `src_instagram`
- VK: `src_vk`
- Личные приглашения: `src_friends`

Потом смотри эффективность через `/sources`.
